Tutoriales PRE-ENTREGA 2

Invernadero de suelo

A continuación, se presentan 4 imágenes que documentan el proceso para generar un modelado 3D de un invernadero de suelo. Esto, comenzando desde un dibujo referencial, hecho en papel, con sus respectivas medidas, para luego, desarrollar el dibujo siguiendo las proporciones y medidas, estableciendo una parametrización de estas, de manera tal de que al momento de variar una de sus medidas, automáticamente se ejecuten las variaciones proporcionales en el resto del modelado.

Una vez generado el componente esencial de este invernadero, se replica 5 veces, para así obtener uno conformado por 6 elementos, con su respectivo techo.

Invernadero de techo

En esta ocasión, se trabajó en llevar una visualización real del invernadero de techo, a un dibujo 2D, para que desde ahí, se generase un objeto en 3D. El principal elemento trabajado fue la utilización de patrones, ya sea para hacerle los orificios para las plantas, como también para poder replicar los elementos dentro de un espacio determinado.
